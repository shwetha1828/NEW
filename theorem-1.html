RANK-NULLITY THEOREM

Statement:

            The Rank-Nullity Theorem states that for any linear transformation (or matrix)

            between finite-dimensional vector spaces, the dimension of the domain is equal to 
 
            the sum of the rank (dimension of the image) and the nullity (dimension of the kernel).

            In simpler terms, the number of columns of a matrix equals the sum of its rank and nullity.

Proof:

            1.Consider a linear transformation 

            T: V -> W between finite-dimensional vector spaces V and W. 

            Assume V has dimension n and W has dimension m. 


            2.Let A be the matrix representation of T. A is an m x n matrix, where n = dim(V). 

            3.Consider the nullspace of T.

            This is the set of all vectors in V that are mapped to the zero vector in W by T (i.e., T(v) = 0).

            The nullspace of T is the kernel of A, denoted as ker(T) or null(A). 

            4.The Rank-Nullity Theorem states: dim(V) = rank(T) + nullity(T). 

            5.Relating to the matrix A:

             This can be rewritten as n = rank(A) + nullity(A), where n is the number of columns in A. 

            6.Proof for the matrix A:
              Case 1:

              If rank(A) = n: 

              The only solution to Ax = 0 is the trivial solution x = 0. 

              Therefore, nullity(A) = 0, and the theorem holds (n = n + 0). 
              
              Case 2:

              If rank(A) = r < n: 

              The row-reduced echelon form of A will have r non-zero rows and n - r zero rows.

              This means there are n - r free variables in the solution to Ax = 0.

              The nullity(A) will be n - r, and the theorem holds (n = r + (n - r)).
